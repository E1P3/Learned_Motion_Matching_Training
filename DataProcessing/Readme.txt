This code is used to process raw bvh files into fbx and binaries that can be used for model training.
The solution requires FBXSDK. To run simply build the project and add libfbxsdk.dll from FBXSKD/lib/vsversion/ to the build folder.

to use other datasets than lafan1 change the files and bone string vectors in FbxToBinConverter.cpp. 

REQUIRES INSTALLING FBXSDK available at teh Autodesk website
